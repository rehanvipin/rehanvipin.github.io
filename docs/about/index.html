Hey, yo.

My name is Rehan, and as you might've guessed, I like to code and solve problems to save time and effort for people.

This is my home page on the internet, so this is going to be all about me.


I like to keep things simple (in general), so I'm trying to make this website with standard HTML elements. That does not mean not using a framework. Frameworks make development simpler by reducing duplication. However, I'm looking for as little magic as possible. I like to understand things and hack around. Astro helps here.

What does that mean? Using semantic HTML elements and using as much of the browser's capabilities as possible. Oh, and no AI. I would like to use my brain when talking about myself, thankyouvermuch.
