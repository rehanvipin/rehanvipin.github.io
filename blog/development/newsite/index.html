Create a portfolio site (minimum expenditure & work)

I created my portfolio site with Hugo, again!

This time, with a plan and adequate free time, I was able to get it to a publishable version 1.0. There is almost no front-end development involved and zero back-end work too(I did work on the backend engine of verless, another static site generator, so not technically zero xD).

I will document the process soon, but here are the main steps:

  • Get a sitemap of your desired portfolio ready
  • Find a theme on Hugo that matches most of your expectations
  • Add as much content across a variety of web pages within the site
  • Modify the theme to suit your needs
  • Create a workflow system to develop on Hugo and host on GitHub pages
  • Buy a domain name and configure it to play nice with Github
    Profit!!
